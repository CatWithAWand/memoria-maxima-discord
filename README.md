## Contributing

### Prerequisites

Ensure you have the following installed:

- Node.js (version 21.7.3 or higher)
- npm (version 10.5.0 or higher)
- Redis

Ensure your local environment is setup properly and you have the latest version of this repository.

Create a new branch and make your changes.

Before commiting your changes, ensure that you have formatted and linted your code by running:

```zsh
npm run format
```

and

```zsh
npm run lint:fix
```

> Note: Ensure that your line endings are set to LF.
